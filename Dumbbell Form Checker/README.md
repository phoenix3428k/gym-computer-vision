# Dumbbell Form Checker & Rep Counter

A Python-based project for checking dumbbell exercise form and counting repetitions from video input.

---

## Features
- **Form Checker:** Analyzes exercise form using computer vision.
- **Rep Counter:** Automatically counts repetitions from video.

## Getting Started

### 1. Fork the Repository
- Click the **Fork** button in the top-right corner.
- Clone your fork:
  ```sh
  git clone https://github.com/YOUR-USERNAME/dumbell.git
  cd dumbell
  ```

### 2. Install Requirements
Make sure you have Python 3.x installed. Install dependencies:
```sh
pip install -r requirements.txt
```

### 3. Run the Project
```sh
python main.py.py video.mp4
```
Replace `video.mp4` with your own video file.
