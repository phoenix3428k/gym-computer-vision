# Gym Computer Vision Project

A collection of AI-powered Computer Vision systems designed for gym training, exercise monitoring, posture correction, and performance tracking. These projects use real-time pose estimation, computer vision, and machine learning techniques to help users train smarter, improve exercise form, and reduce injury risks.

---

## Projects

### 1. Dumbbell Form Checker & Rep Counter

An AI-powered workout assistant that analyzes dumbbell exercises from video footage, evaluates exercise technique, and automatically counts repetitions.

#### Key Features
- Exercise form analysis
- Automatic rep counting
- Motion tracking
- Workout performance monitoring
- Real-time visual feedback

#### Applications
- Home workouts
- Personal training
- Gym exercise monitoring
- Fitness coaching

---

### 2. Plank Form Checker & Hold Timer

A computer vision system that evaluates plank posture using pose estimation. It tracks body alignment, identifies common form mistakes, and measures hold duration.

#### Key Features
- Real-time posture analysis
- Body alignment detection
- Hold timer tracking
- Injury-risk warnings
- Visual feedback overlays

#### Applications
- Core training
- Physical therapy
- Sports conditioning
- Fitness assessment

---

### 3. Squant Form Checker & Hold Timer

A posture analysis system focused on squat and plank-style body positioning. The project evaluates movement quality and stability while providing real-time feedback.

#### Key Features
- YOLOv8 pose estimation
- Posture quality assessment
- Hold-duration tracking
- Movement monitoring
- Cinematic HUD visualization

#### Applications
- Squat training
- Strength conditioning
- Mobility assessment
- Athletic performance improvement
